📚 시작하기

교회 헌금관리시스템은 한인교회의 효율적인 재정 관리와 교인 관리를 위해 특별히 설계된 종합 관리 솔루션입니다.

💡 이 시스템으로 할 수 있는 일

  • ✓ 교인 정보를 체계적으로 관리
  • ✓ 헌금 내역을 정확하게 기록
  • ✓ 다양한 보고서로 교회 재정 현황 파악
  • ✓ 교인별 헌금 패턴 분석으로 목회적 돌봄
  • ✓ 연간 세금 보고를 위한 자료 준비

주요 특징

🏢 교회별 독립 시스템

각 교회의 데이터는 완전히 분리되어 안전하게 관리됩니다.

📱 모바일 지원

스마트폰, 태블릿, 컴퓨터 모든 기기에서 사용 가능합니다.

🔒 보안

암호화된 연결과 자동 로그아웃으로 정보를 보호합니다.

📊 실시간 통계

대시보드에서 교회 현황을 한눈에 파악할 수 있습니다.

💾 설치 및 접속

웹 브라우저로 접속하기

  1. 1

    웹 브라우저 열기

    Chrome, Firefox, Safari, Edge 등 최신 브라우저 사용 권장

  2. 2

    시스템 주소 입력

    제공받은 URL을 주소창에 입력

  3. 3

    로그인

    교회 로그인 ID와 패스워드 입력

모바일 앱처럼 설치하기 (PWA)

📱 iPhone/iPad

  1. 1. Safari로 시스템 접속
  2. 2. 하단 공유 버튼 탭
  3. 3. "홈 화면에 추가" 선택
  4. 4. "추가" 탭

🤖 Android

  1. 1. Chrome으로 시스템 접속
  2. 2. 우측 상단 메뉴(⋮) 탭
  3. 3. "홈 화면에 추가" 선택
  4. 4. "추가" 탭

📊 대시보드

대시보드는 교회의 현황을 한눈에 파악할 수 있는 종합 현황판입니다.

📊 주요 통계 카드

  • • 이번 달 총 헌금액
  • • 전월 대비 증감률
  • • 등록 교인 수
  • • 오늘의 헌금 현황

📈 차트와 그래프

  • • 월별 헌금 추이 그래프
  • • 헌금 종류별 비율 차트
  • • 최근 6개월 트렌드
  • • 연간 성장 지표

👥 교인 관리

교인 등록하기

  1. ✅ 기본 정보 입력 - 이름, 전화번호, 주소 등
  2. ✅ 직분 선택 - 목사, 장로, 권사, 집사 등
  3. ✅ 직분 상태 설정 - 시무, 은퇴, 협동 등

💡 목회 활용 팁

교인 상세 페이지에서 헌금 내역을 확인하여 신앙생활의 변화를 파악하고, 필요한 목회적 돌봄을 제공할 수 있습니다.

💰 헌금 관리

📝 일반 등록

  • • 교인 선택 또는 비교인 입력
  • • 헌금 종류 선택
  • • 금액 입력
  • • 날짜 및 메모 추가

⚡ 빠른 등록

  • • 자주 사용하는 헌금 종류
  • • 간단한 입력 인터페이스
  • • 모바일 최적화
  • • 연속 입력 가능

🔐 보안 설정

패스워드 변경 및 접근 권한

❤️ 목회적 활용 방안

💡 이 시스템은 단순한 헌금 기록을 넘어 목회적 돌봄의 도구로 활용될 수 있습니다

데이터는 차가운 숫자가 아니라 교인들의 신앙 여정을 보여주는 따뜻한 이야기입니다.

🙏 신앙 여정 이해

개인내역 보고서를 통해 교인의 신앙생활 패턴과 변화를 이해

💝 목회적 관심

헌금 패턴 변화를 통해 적절한 목회적 돌봄 제공

📈 교회 성장 분석

연간 보고서로 교회의 성장과 변화 분석

📅 효과적인 계획

헌금 추이 파악으로 사역과 재정 계획 수립

"다양한 보고서와 통계 기능은 교인들의 삶을 더 깊이 이해하고, 교회 공동체를 더 건강하게 세워가는 데 도움이 됩니다."

❓ 자주 묻는 질문

데이터는 안전한가요?

네, 모든 데이터는 암호화되어 저장되며, 각 교회의 데이터는 완전히 분리되어 관리됩니다.

여러 명이 동시에 사용할 수 있나요?

네, 여러 관리자가 동시에 접속하여 작업할 수 있으며, 실시간으로 데이터가 동기화됩니다.

모바일에서도 사용할 수 있나요?

네, 스마트폰, 태블릿 등 모든 기기에서 사용 가능합니다.

연말 정산용 자료를 출력할 수 있나요?

네, 교인별 연간 헌금 내역서를 생성하여 출력하거나 PDF로 저장할 수 있습니다.

도움이 필요하신가요?

시스템 사용 중 문제가 발생하거나 추가 도움이 필요하시면 언제든지 연락주세요.
